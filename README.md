# 🎡 Random Flag Wheel — ICST 2026

A fully animated, interactive flag-based quiz game built for a corporate event.
Players spin a dynamic flag carousel, guess the country, and compete for high scores in real-time.

---

## 🌐 Live Demo

👉 **Play Now:** https://flag-game-amex.web.app/

---

## 🚀 Features

* 🎡 Animated flag spinner (carousel style)
* 🌍 Real country flags using CDN
* 🎯 Multiple choice quiz system
* 🎉 Confetti + sound effects
* ⏱️ 45-second countdown timer
* 🏆 Score tracking
* 👤 Player name input
* 📊 Leaderboard system
* 🖥️ Fullscreen mode (event-ready)
* 📱 Fully responsive design

---

## 🛠️ Tech Stack

* React (Vite)
* TypeScript
* Tailwind CSS
* Framer Motion
* Howler.js (sound)
* Canvas Confetti

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/ANUJSELF/flag-game.git
cd flag-game
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

---

## 🏗️ Build for Production

```bash
npm run build
```

---

## 🌐 Deployment (Firebase)

1. Install Firebase CLI:

```bash
npm install -g firebase-tools
```

2. Login:

```bash
firebase login
```

3. Initialize:

```bash
firebase init
```

Select:

* Hosting

Options:

* Public directory: `dist`
* Single Page App: Yes
* Overwrite index.html: No

4. Deploy:

```bash
firebase deploy
```

## 🎯 How to Play

1. Enter your name
2. Click **Spin**
3. Watch the flag animation
4. Guess the correct country
5. Score as many correct answers as possible before time runs out
